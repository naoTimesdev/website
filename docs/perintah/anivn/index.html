Anilist & VNDB

Merupakan perintah untuk parsing dan memberikan kembali user informasi dari sebuah Anime/Manga atau VN
Setiap command support pagination agar bisa melihat berbagai macam hasil dengan ngeklik tombol reactions yang disediakan oleh Bot.

Anilist#

Nama CommandPenjelasanContohAlias
!anime [judul]Mencari/melihat anime.
[judul]: Judul anime ataupun manga yang ada di Anilist.co		!anime hitoribocchi!animu
!kartun
!ani
!manga [judul]Mencari/melihat manga.
[judul]: Judul anime ataupun manga yang ada di Anilist.co		!anime hitoribocchi!mango
!komik
!tayangMelihat jadwal tayang anime pada musim sekarang!tayang-
Tambahan

⏪ Anime/Manga Sebelumnya
⏩ Anime/Manga Selanjutnya
✅ Selesai melihat (Menghapus embed)
⏳ Waktu ke episode selanjutnya (Untuk !anime saja)
👍 Melihat info anime kembali (Untuk !anime saja, muncul setelah memakai ⏳)

Tambahan untuk !tayang

0 - H: Melihat listing per sisa hari ke episode selanjutnya
✅ Selesai melihat/kembali ke listing
⏳ Waktu ke episode selanjutnya (Untuk !anime saja)
👍 Melihat info anime kembali (Untuk !anime saja, muncul setelah memakai ⏳)

⏩ VN Selanjutnya
📸 Melihat screeenshot dari VN (Jika ada)
✅ Melihat info VN kembali (Muncul setelah memakai 📸)
